## 문제

## 입력
<p>첫째 줄에 N과 K가 주어진다. (1 ≤ K < N ≤ 500,000)</p>

<p>둘째 줄에 N자리 숫자가 주어진다. 이 수는 0으로 시작하지 않는다.</p>

## 출력
<p>입력으로 주어진 숫자에서 K개를 지웠을 때 얻을 수 있는 가장 큰 수를 출력한다.</p>
