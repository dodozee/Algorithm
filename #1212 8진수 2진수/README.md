## 문제
<p>8진수가 주어졌을 때, 2진수로 변환하는 프로그램을 작성하시오.</p>

## 입력
<p><span style="line-height:1.6em">첫째 줄에 8진수가 주어진다. 주어지는 수의 길이는 333,334을 넘지 않는다.</span></p>

## 출력
<p>첫째 줄에 주어진 수를 2진수로 변환하여 출력한다. 수가 0인 경우를 제외하고는 반드시 1로 시작해야 한다.</p>
